# procuration-macron.fr

Plateforme de mise en relation entre citoyens souhaitant faire procuration et citoyens pouvant voter en leur nom

## Backend doc
See Makefile

## Front doc
### Requirements
- Node.js (v6.9.5)
- npm (3.10.10)


### First install
```bash
$ npm install
```


### Watch
```bash
$ gulp
```


### Build
```bash
$ gulp build
```
